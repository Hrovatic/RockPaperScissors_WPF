# RockPaperScissorsWpf

A Rock-Paper-Scissors desktop game built with C# and WPF.

## Features

- WPF UI
- Play against a computer opponent
- Game result tracking logic
- Expandable structure for difficulty modes and additional features

## Tech Stack

- C#
- WPF (.NET)

## Run

From repository root:

```bash
dotnet run --project KamenŠkarjePapir/KamenŠkarjePapir.csproj
```
